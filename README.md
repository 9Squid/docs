# Buildmarkets Developer Docs

Documentation for the [Buildmarkets](https://buildmarkets.com) platform — build, launch, and scale compliant investment products.

Covers API references, SDKs, guides for self-directed investing, wealth management, FIX gateway, and more.

**Live docs:** [docs.buildmarkets.com](https://docs.buildmarkets.com)

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview changes locally:

```bash
npm i -g mint
```

Run the dev server:

```bash
mint dev
```

Preview at `http://localhost:3000`.

## Publishing

Changes pushed to the default branch deploy automatically via the [Mintlify GitHub app](https://dashboard.mintlify.com/settings/organization/github-app).

## Troubleshooting

- Dev server not starting — run `mint update` to get the latest CLI.
- 404 on a page — make sure you're in the folder with `docs.json`.
